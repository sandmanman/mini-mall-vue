# A shopping cart app.

This is lab project for learning about
- Vue
- Vuex
- Vuex Subscribe LocalStorage

# Screenshot
![Cart](http://i.imgur.com/L1zcEnB.png)

# Features
- Read products from json
- Add to cart
- Update cart item
- Remove cart item
- Summary price

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# run unit tests
npm run unit

# run e2e tests
npm run e2e

# run all tests
npm test
```
